# Task Manager: uses projects & tasks philosophy 

This application is for
[*RubyGarage*](https://rubygarage.com.ua/#about-course/)
by [*Maxym Kosenko*](mailto:maxie7@protonmail.com).

Functional requirements:
- you can create/update/delete projects
- you can add tasks to your project(s)
- you can update & delete tasks
- All tasks are sortable(and draggable), besides you can also drag & sort you Projects
- You can add a deadline to your task(s), but it's optional. If you don't do it will be marked as "no deadline"
- Tasks could be checked as 'done' by striking them out.

It has user authentication solution
